## 🚀 快速开始

### 方式一：在线预览（推荐）

1. 在浏览器中打开 `api-docs-preview.html`
2. 文档会自动加载并渲染
3. 使用右侧目录导航快速跳转

```bash
# 在项目根目录下
open api-docs-preview.html  # macOS
xdg-open api-docs-preview.html  # Linux
start api-docs-preview.html  # Windows
```

### 方式二：使用Markdown阅读器

推荐的Markdown阅读器：
- **VS Code**: 内置Markdown预览（Ctrl/Cmd + Shift + V）
- **Typora**: 专业的Markdown编辑器
- **MacDown**: macOS专用
- **MarkText**: 跨平台开源编辑器

### 方式三：在线平台

将文档上传到以下平台：
- **GitHub**: 自动渲染Markdown
- **GitBook**: 生成在线文档站点
- **Notion**: 导入Markdown文件
- **语雀**: 支持Markdown导入

## 📖 阅读建议

### 新手入门

1. **第一步**: 阅读"API文档摘要.md"
   - 了解文档结构
   - 查看接口统计
   - 理解核心特性

2. **第二步**: 阅读"通用规范"章节
   - 请求格式
   - 响应格式
   - 错误处理

3. **第三步**: 阅读"认证授权"章节
   - 登录流程
   - Token使用
   - 权限控制

4. **第四步**: 根据需求查找具体接口
   - 使用目录快速定位
   - 查看请求示例
   - 理解响应格式

### 开发人员

1. **查找接口**: 使用Ctrl/Cmd + F搜索关键词
2. **复制示例**: 直接复制请求/响应示例
3. **参考代码**: 查看"最佳实践"章节的代码示例
4. **错误处理**: 参考"错误码说明"章节

### 测试人员

1. **了解接口**: 查看接口列表和参数说明
2. **准备数据**: 参考"数据字典"章节
3. **测试用例**: 根据请求示例编写测试用例
4. **验证响应**: 对照响应格式验证返回数据

## 💡 使用技巧

### 1. 搜索技巧

在Markdown文档中搜索：
- 搜索接口路径: `/api/resumes`
- 搜索HTTP方法: `POST`、`GET`
- 搜索错误码: `VALIDATION_ERROR`
- 搜索字段名: `phone`、`name`

### 2. 代码复制

所有代码块都可以直接复制使用：
```javascript
// 示例：登录请求
const response = await axios.post('/api/auth/login', {
  username: 'admin',
  password: 'password123'
});
```

### 3. 书签管理

在浏览器中为常用章节添加书签：
- 认证授权: `#3-认证授权`
- 简历管理: `#4-简历管理`
- 错误码: `#10-错误码说明`
