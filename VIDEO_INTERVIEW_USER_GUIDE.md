# 📹 视频面试功能使用指南

## 🎯 功能介绍

视频面试功能支持 **3-6 人同时在线视频面试**，适用于家政人员面试场景。

### ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🎥 **多人视频** | 支持 3-6 人同时在线 |
| ✨ **美颜功能** | 内置多档位美颜调节 |
| 🖥️ **屏幕共享** | 可展示简历、文档等 |
| 💬 **文字聊天** | 实时文字交流 |
| 👥 **成员管理** | 查看在线成员、踢出成员 |
| 🎤 **音视频控制** | 开关摄像头、麦克风 |
| 📐 **自动布局** | 根据人数自动调整布局 |

## 🚀 快速开始

### 1. 访问视频面试

**方式一：通过菜单**
1. 登录系统：https://crm.andejiazheng.com
2. 点击左侧菜单 **"视频面试"**

**方式二：直接访问**
- 直接访问：https://crm.andejiazheng.com/interview/video

### 2. 创建房间（面试官）

1. 进入视频面试页面
2. 点击 **"生成新房间号"** 按钮
3. 系统自动生成房间号，例如：`room_1730883159_abc123`
4. 输入您的名称（默认使用登录用户名）
5. 点击 **"加入视频面试"** 按钮

### 3. 邀请参与者

**将房间号分享给其他参与者**：

**方式一：微信分享**
```
【视频面试邀请】
房间号：room_1730883159_abc123
请登录系统后，进入"视频面试"页面，输入此房间号加入面试。
```

**方式二：电话告知**
- 直接电话告知房间号
- 对方登录后输入房间号即可加入

### 4. 加入房间（参与者）

1. 登录系统
2. 进入"视频面试"页面
3. 在"房间号"输入框中输入收到的房间号
4. 输入您的名称
5. 点击 **"加入视频面试"** 按钮

## 🎮 功能使用

### 基础控制

#### 开关摄像头 🎥
- 位置：视频界面底部
- 图标：摄像头图标
- 功能：点击可开启/关闭摄像头

#### 开关麦克风 🎤
- 位置：视频界面底部
- 图标：麦克风图标
- 功能：点击可开启/关闭麦克风

#### 文字聊天 💬
- 位置：视频界面右侧或底部
- 图标：聊天气泡图标
- 功能：发送文字消息

#### 成员列表 👥
- 位置：视频界面右侧或底部
- 图标：人员图标
- 功能：查看在线成员

### 高级功能

#### 美颜设置 ✨

1. 点击视频界面的 **"设置"** 按钮（齿轮图标）
2. 找到 **"美颜"** 选项
3. 调节美颜参数：
   - **磨皮**：平滑皮肤
   - **美白**：提亮肤色
   - **大眼**：放大眼睛
   - **瘦脸**：瘦脸效果

**提示**：
- 建议在光线充足的环境下使用
- 美颜效果可实时预览
- 可随时调整参数

#### 屏幕共享 🖥️

**使用场景**：
- 展示家政人员简历
- 展示工作证明文件
- 展示培训证书

**操作步骤**：
1. 点击底部 **"屏幕共享"** 图标
2. 选择要共享的窗口或屏幕
3. 点击 **"共享"** 按钮
4. 其他参与者可以看到您的屏幕
5. 再次点击图标可停止共享

#### 踢出成员 🚪

**权限**：仅房主（第一个进入房间的人）可用

**操作步骤**：
1. 点击 **"成员列表"** 图标
2. 找到要踢出的成员
3. 点击该成员旁边的 **"踢出"** 按钮
4. 确认操作

**使用场景**：
- 移除不相关的参与者
- 处理干扰面试的人员

#### 布局切换 📐

**可用布局**：
- **自动布局**：根据人数自动调整
- **网格布局**：所有人平铺显示
- **演讲者布局**：突出显示当前发言者

**操作步骤**：
1. 点击底部 **"布局"** 图标
2. 选择想要的布局方式

### 离开房间

**方式一：点击按钮**
- 点击右上角红色 **"离开房间"** 按钮
- 确认离开

**方式二：关闭页面**
- 直接关闭浏览器标签页
- 系统自动退出房间

## 📱 使用场景

### 场景 1：家政人员面试

**参与者**：
- HR（张经理）- 房主
- 客户（李女士）
- 家政人员（王阿姨）

**流程**：

1. **准备阶段**
   - 张经理提前登录系统
   - 创建视频面试房间
   - 通过微信将房间号发送给李女士和王阿姨

2. **面试开始**
   - 李女士和王阿姨分别登录并加入房间
   - 三人开启摄像头和麦克风
   - 王阿姨可以开启美颜功能

3. **面试过程**
   - 张经理提问，王阿姨回答
   - 李女士补充问题
   - 如需展示王阿姨的简历，张经理可以使用屏幕共享
   - 使用文字聊天记录重要信息

4. **面试结束**
   - 所有人点击"离开房间"
   - 张经理在系统中记录面试结果

### 场景 2：多对一面试

**参与者**：
- HR 团队（3人）
- 候选人（1人）

**流程**：
1. HR 主管创建房间
2. 邀请其他 HR 和候选人加入
3. 4人同时在线面试
4. 轮流提问
5. 使用文字聊天记录评价
6. 面试结束后保存记录

### 场景 3：远程培训

**参与者**：
- 培训师（1人）
- 家政人员（5人）

**流程**：
1. 培训师创建房间
2. 邀请家政人员加入
3. 培训师使用屏幕共享展示培训材料
4. 家政人员可以提问
5. 使用文字聊天记录要点

## ⚙️ 设置建议

### 网络要求

| 参与人数 | 建议带宽 | 最低带宽 |
|---------|---------|---------|
| 2-3 人 | 2 Mbps | 1 Mbps |
| 4-5 人 | 4 Mbps | 2 Mbps |
| 6 人 | 6 Mbps | 3 Mbps |

### 设备要求

**电脑**：
- CPU：Intel i3 或以上
- 内存：4GB 或以上
- 摄像头：720P 或以上
- 麦克风：内置或外接

**手机/平板**：
- 系统：iOS 12+ / Android 8+
- 内存：2GB 或以上
- 前置摄像头

### 浏览器要求

**推荐浏览器**：
- ✅ Chrome 74+
- ✅ Edge 79+
- ✅ Safari 13+
- ✅ Firefox 66+

**不支持**：
- ❌ IE 浏览器

### 环境建议

**光线**：
- 面部光线充足
- 避免逆光
- 避免强光直射

**背景**：
- 整洁的背景
- 避免杂乱
- 可使用虚拟背景（如支持）

**声音**：
- 安静的环境
- 避免回音
- 建议使用耳机

## 🐛 常见问题

### Q1: 无法加入房间？

**可能原因**：
- 房间号输入错误
- 网络连接问题
- 浏览器不支持

**解决方法**：
1. 检查房间号是否正确
2. 检查网络连接
3. 更换浏览器（推荐 Chrome）
4. 刷新页面重试

### Q2: 看不到对方视频？

**可能原因**：
- 对方未开启摄像头
- 网络延迟
- 浏览器权限未授予

**解决方法**：
1. 确认对方已开启摄像头
2. 检查浏览器是否允许摄像头权限
3. 刷新页面重新加入

### Q3: 听不到对方声音？

**可能原因**：
- 对方未开启麦克风
- 本地音量设置过低
- 浏览器权限未授予

**解决方法**：
1. 确认对方已开启麦克风
2. 检查本地音量设置
3. 检查浏览器是否允许麦克风权限

### Q4: 视频卡顿？

**可能原因**：
- 网络带宽不足
- 参与人数过多
- 设备性能不足

**解决方法**：
1. 降低视频分辨率（在设置中调整）
2. 关闭不必要的摄像头
3. 减少参与人数
4. 关闭其他占用网络的程序

### Q5: 美颜效果不明显？

**解决方法**：
1. 点击"设置"按钮
2. 找到"美颜"选项
3. 调高美颜参数
4. 确保光线充足

### Q6: 屏幕共享失败？

**可能原因**：
- 浏览器权限未授予
- 系统权限限制

**解决方法**：
1. 检查浏览器是否允许屏幕共享权限
2. 在系统设置中允许浏览器录制屏幕（Mac）
3. 重新选择要共享的窗口

**建议**：
