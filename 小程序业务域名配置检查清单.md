## 🔧 常见问题

### Q1: 业务域名已添加，但验证失败

**原因**：爱签服务器上没有校验文件

**解决**：
1. 联系爱签客服
2. 提供 AppID: 141496759
3. 提供校验文件
4. 等待爱签客服上传

### Q2: 如何确认校验文件是否已上传？

在浏览器中访问：
```
https://hxcx.asign.cn/WxVerifyFile.txt
```

- 如果能下载文件 → 已上传 ✅
- 如果显示404 → 未上传 ❌

### Q3: 验证通过后，小程序还是无法访问

**可能原因**：
1. 小程序缓存问题 → 重新编译小程序
2. 域名配置未生效 → 等待5-10分钟
3. 小程序版本问题 → 重新上传小程序代码

### Q4: 开发者工具可以访问，真机无法访问

**原因**：开发者工具勾选了"不校验域名"

**解决**：
1. 取消勾选"不校验合法域名"
2. 在开发者工具中测试
3. 如果报错，说明域名配置有问题
