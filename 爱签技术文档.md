# 爱签电子合同技术文档

## 一、概述

爱签（Asign）是第三方电子合同签约平台，提供完整的电子签名服务。

- **官网**: https://www.asign.cn
- **测试环境**: https://prev.asign.cn
- **生产环境**: https://oapi.asign.cn

## 三、核心业务流程

### 3.1 签署流程

```
1. 上传合同文件 / 获取模板
      ↓
2. 创建合同 (createContract)
      ↓
3. 添加签署人 (addSigner) + 设置签章位置
      ↓
4. 获取签署链接 (返回给用户)
      ↓
5. 用户签署 → 回调通知
      ↓
6. 下载合同 (downloadContract)
```

### 3.2 合同下载（支持多格式）

```typescript
downloadFileType: number;
// 1: PDF文件
// 2: 多个单张PNG，含PDF
// 3: 分页PNG压缩，含PDF  
// 4: 合同单张图片，不含PDF
// 5: 所有分页图片，不含PDF
```

## 五、签名算法

### 请求头格式

```
POST {host}/v1/contract/createContract
Content-Type: application/json; charset=UTF-8
X-Tsign-Open-Token: {token}  # OAuth2模式需要
sign: {RSA签名}
```

### 签名步骤

1. **bizData 排序**：按 ASCII 码升序排列 Key
2. **生成待签串**：`CanonicalString = bizData + MD5(bizData) + appId + timestamp`
3. **RSA 签名**：使用 SHA1 + 私钥加密
4. **Base64 编码**

## 八、与芝麻背调集成

### 场景：授权书签字页用于背调

当候选人签署合同后，可获取签字页用于芝麻背调授权：

```typescript
// 1. 下载合同（转图片格式）
const result = await esign.downloadSignedContract(contractNo, {
  downloadFileType: 5  // 所有分页图片
});

// 2. 取最后一页（签字页）
const signPageImage = result.images[result.images.length - 1];

// 3. 上传到芝麻背调
const uploadResult = await zmdb.uploadImage(signPageImage);

// 4. 发起背调（授权书模式）
await zmdb.queryReport({
  authType: '3',  // 授权书授权
  authStuffUrl: uploadResult.imageUrl,
  // ...
});
```
